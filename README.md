# איפה בעולם? — משחק גאוגרפיה בעברית

משחק דפדפן בעברית (RTL) בהשראת משחקי גאוגרפיה וחקירה קלאסיים. אתם בלשים בסוכנות דלתא שרודפים אחר חברי ארגון הצל ברחבי העולם.

## התקנה והרצה

```bash
npm install
npm run dev
```

פתחו את הכתובת שמוצגת בטרמינל (בדרך כלל `http://localhost:5173`).

## בנייה לפרודקשן

```bash
npm run build
npm run preview
```

## איך משחקים

1. **התראת פשע** — גניבה דווחה בעיר מסוימת.
2. **חקור** — בקרו ב-3 מקומות בכל עיר כדי לאסוף רמזים מעדים.
3. **ספר העולם** — השתמשו במידע על מדינות (שפה, מטבע, אתרים) כדי לפענח רמזים.
4. **טוס** — טוסו לעיר הבאה; מסלול נכון ממשיך את המרדף, מסלול שגוי מבזבז זמן.
5. **אינטרפול** — סננו חשודים לפי תכונות עד שמתקבל צו מעצר.
6. **מעצר** — בעיר האחרונה, חקור את המקום הנכון עם צו מעצר תקף.

## מכניקת זמן

- זמן התחלתי: 36 יחידות
- חקירה: −1
- טיסה נכונה: −1
- טיסה שגויה: −3

## ממשק Deluxe

המשחק מושפע מ-[Where in the World is Carmen Sandiego? Deluxe Edition](https://www.mobygames.com/game/8195/where-in-the-world-is-carmen-sandiego-deluxe-edition/):

- **מסך מפוצל** — תמונת מיקום גדולה + פאנל מידע (כמו VGA 640×480)
- **שיחת וידאו עם המפקדת** — בריifings בזמן אמת
- **סרגל פעולות** — חקור · טוס · CrimeNet · ספר עולם
- **מפת עולם** — בחירת יעדים עם סיכות על גlobe
- **CrimeNet** — תיקי חשודים עם תמונות וסינון
- **תמונות Unsplash** — ערים, מקומות, חשודים ואוצרות

## תוכן המשחק

- **22 ערים** ברחבי העולם עם קשרי טיסה
- **22 מדינות** בספר העולם
- **7 חשודים** עם תמונות פיקסל
- **10 סוגי אוצרות** עם איורים
- **גרפיקת פיקסל retro** בסגנון CGA — ערים, מקומות, עדים, חשודים ואוצרות

## טכנולוגיות

- React 19 + TypeScript + Vite
- מנוע משחק טהור (ללא תלות ב-React) ב-`src/game/`
- נתוני משחק בעברית ב-`src/data/gameData.ts`

## הערות

המשחק משתמש בשמות ודמויות מקוריים (סוכנות דלתא, ארגון הצל) ואינו משתמש בזכויות יוצרים של המותג המקורי.
